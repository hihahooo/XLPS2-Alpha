# 文档② 布局（Layout）

> 八文档之一。权威约束见 `../pcb/stackup.md` 与 `../pcb/layout.md`。本文为布局审查依据。

## 1. 板型与层叠

- 2 层 PCB，100×80mm，1.6mm FR-4，ENIG；**禁用 4 层，无内层平面**（硬件约束硬指标）。
- Top=信号/局部电源/地铜皮；Bottom=信号/主地回流铜皮。

## 2. 分区（与 `pcb/layout.md` 一致）

- 上：电源链（J1/L30/Q10/F1/U31/U30）→ 经 ≥4mm 分割槽进入次级。
- 右：接口区（ETH+J14、ISO1042+RS485+J2~J5、B0505S）。
- 中：MCU + I2C 存储 + WS63。
- 下：安全回路（U9/Q11/K1）+ 指示 + 调试（J6/J13）。

## 3. 布局硬规则

1. 电源链沿板边一字排开，U30 下方留 ≥4mm 净空（初/次级不相交铜）。
2. STM32H743 居中，100nF 退耦 <5mm，VCAP 紧靠；Y1 紧靠 PH0/PH1 并包地。
3. RMII 50MHz 与 SDIO 走线等长包地；WS63 天线净空 ≥5mm 禁铺铜。
4. 安全回路路径最短，K1 大电流走线不穿 MCU 区。

## 4. 布线规则摘要

- 24V 通流 ≥2mm 宽/露铜；5V ≥0.4mm；3.3V ≥0.3mm；过流处多过孔。
- CAN/RS485 差分紧耦合，TVS 最靠近端口；终端 120Ω 近连接器。
- 地回流经 Bottom 铜皮+过孔阵；信号禁止跨地分割（隔离边界除外）。

## 5. 审查清单

- [ ] 板尺寸 100×80 ±0.2mm；2 层确认无内层
- [ ] 初/次级爬电 ≥4mm
- [ ] 隔离收发器隔离地分割 ≥2mm
- [ ] 测试点位置与 `06_testpoints.md` 一致
- [ ] DRC 0 违规；与原理图 netlist 比对一致
