# 文档⑤ 装配（Assembly）

> 八文档之一。装配顺序与 DFM 要点；与 `03_bom.md` 物料、`04_silkscreen.md` 位号一致。

## 1. 装配顺序（样机手工/SMT）

1. **电源链先装**：J1 → L30 → Q10 → F1 → U31 → U30（隔离边界最先确立，便于单测）。
2. **次级电源**：U33/U34 + 输出电容 → 上电测 NET_5V/NET_3V3。
3. **MCU 与最小系统**：U1 + Y1/Y2 + 复位 + SWD（先不焊接口，避免短路干扰调试）。
4. **接口**：ISO1042/ADM2483 + B0505S + LAN8742A + J14 + J2~J5。
5. **无线与存储**：U5 WS63 + U6/U7 I2C。
6. **安全与指示**：U9/Q11/K1 + LED/BUZ1 + BT1。

## 2. DFM 要点

- LQFP176 钢网开窗 1:1；QFN24（U4）露窗防桥连，底部散热 pad 接地过孔阵列。
- B0505S 为 SIP4，注意引脚 1 标识；隔离侧悬空不连逻辑地。
- 所有 0603 被动件用 0.12mm 钢网；钽电容极性核对。
- 工装：首件 ICT（测试点见 `06_testpoints.md`）。

## 3. 检验（装配后）

- [ ] 目视：无反向、无桥连、隔离区无铜连接
- [ ] 通断：NET_GND_IN 与 NET_GND 间阻值 ≥100MΩ（隔离完好）
- [ ] 上电：J1 加 24V（限流源），测 NET_24V_BOARD/5V/3V3 正确
- [ ] HW-PWR-001~005 全过（见 `06_testpoints.md`）

## 4. 审查清单

- [ ] 装配顺序与丝印/位号一致
- [ ] 隔离边界装配后无意外短接
- [ ] 首件 ICT 通过
