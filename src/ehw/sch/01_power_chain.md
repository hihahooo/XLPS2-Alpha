# 24V 隔离电源链（逐级设计与验证）

> 权威链序（全文档一致，绝对不可颠倒）：
> `J1 → L30 → Q10 → F1 → U31 → U30 → NET_24V_BOARD`

```
J1(DC24V_IN)
 └─[L30 π/共模滤波]→ Q10(防反 P-MOS AO3401A, 12V Zener 钳 Vgs)
    └─[F1 PPTC 2A]→ U31(eFuse TPS26600, 可复位 OCP/SCP)
       └─[U30 隔离 DC-DC 金升阳 URB2424P-30WR3 ≥1.5kV]
          └─ NET_24V_BOARD ──► Buck(U33 24→5V)→LDO(U34 5→3.3V) 及 传感器/LED/蜂鸣器
```

地分割：初级 `NET_GND_IN`（U30 前）与次级 `NET_GND`（U30 后）之间 ≥4mm 爬电距离。

---

## 第 1 级 J1 — 24V 电源输入

| 项 | 值 |
|----|-----|
| 器件 | J1，2-pin 端子（5.08mm 间距，带锁扣） |
| 引脚 | J1-1 = +24V_IN，J1-2 = GND_IN |
| 额定 | 24Vdc，≤3A；反接/浪涌由后级处理 |
| 防护前级 | D30（SMBJ24A TVS，钳位输入浪涌至 ~38V）跨接在 J1 两端 |

**机理**：物理输入接口；TVS D30 吸收静电/感性关断尖峰，为后级争取余量。

## 第 2 级 L30 — 共模/π 滤波

| 项 | 值 |
|----|-----|
| 器件 | L30，共模电感 10µH / 3A（屏蔽工字/环形） |
| 接法 | 串入 +24V 主线；两侧各并 10µF + 0.1µF（C_L30a/C_L30b） |
| 作用 | 抑制来自 48V 动力线耦合的共模噪声，降低对隔离前的传导干扰 |

**机理**：π 型 EMI 滤波，改善 CE（传导发射）与对伺服驱动噪声的抗扰。

## 第 3 级 Q10 — 防反接（理想二极管）

| 项 | 值 |
|----|-----|
| 主器件 | Q10 = AO3401A（P-MOS，SOT23，-30V/-4A，Vgs±12V） |
| 钳位 | D31 = BZT52C12（12V Zener，SOD123）串 R10(10k) 接 GND_IN |
| 偏置 | R11(100k) 从 +24V 拉栅极；正向导通时 Vgs ≈ -(24-12) = -12V（恰在额定内） |

**机理（防反）**：
- 正向：源极=+24V，栅极经 D31 钳到约 12V 低于源 → Vgs≈-12V < Vth(-1V) → 导通，近似零压降。
- 反向（24V 与 GND 接反）：源极被拉到负压，栅极=0 → Vgs>0 → 关断，体二极管反偏，负载完全隔离，零损坏。
- AO3401A 的 Vds=-30V > 24V 裕量充足；D31 保证 Vgs 不超 ±12V 额定。

**验证点**：HW-PWR-001（防反接后板载无电压、无发热、恢复后正常）。

## 第 4 级 F1 — 可恢复保险（PPTC）

| 项 | 值 |
|----|-----|
| 器件 | F1 = PPTC 自恢复保险，保持 2A / 跳闸 ~4A（1812） |
| 作用 | 针对 Q10 之后的硬短路/严重过流的**第一道**粗保护，避免后级 eFuse  alone 承受极端电流 |

**机理**：过流发热阻值骤升限流；故障解除冷却后自恢复。与 U31 形成「粗+精」双重保护。

## 第 5 级 U31 — eFuse 可复位过流/短路保护

| 项 | 值 |
|----|-----|
| 器件 | U31 = TPS26600PW（60V/2A 工业 eFuse，TSSOP16，内置反向输入保护） |
| 限流 | **R_ILIM = 6.04kΩ(1%)** → I_LIM ≈ 11.8/6.04 ≈ 1.95A（≈2A，见下注） |
| 模式 | MODE 经 402kΩ→RTN：主动限流 + 自动重试（auto-retry） |
| UVLO/OVP | UVLO/OVP 直接接 RTN → 内部默认 UV=15V / OV=33V（适配 24V 系统） |
| 指示 | FLTb 开漏 → 上拉至 NET_3V3 → MCU PB0（故障中断）；PGOODb → 上拉 → MCU PB1 |
| 输出 | C31 = 22µF 陶瓷，缓启动由外部电容设定 |

> ✅ **已裁决确认（主理人 #2）**：原 `docs/modules/ehw.md` 写「R_ILIM=24k ~2A」。
> 按 TI TPS26600 数据手册公式 **I_LIM(A)=11.8/R_ILIM(kΩ)**，24kΩ 实际仅 ≈0.49A，
> 与「2A」目标矛盾。主理人裁定目标限流 ≈2A，采纳 **6.04kΩ(1%)** 实现 ≈2A 限流，
> 本设计维持，无需改动；`docs/modules/ehw.md` §3 文字已同步更正。

**机理**：
- 过载：主动限流调节至 I_LIM，FLTb 在 875µs 后拉低；若持续致热关断，512ms 后自动重试。
- 短路：同样限流并快速重试/锁存（本设计选自动重试，适合现场可自愈工况）。
- 反向输入：TPS26600 内置反向极性保护，与 Q10 形成正交冗余（Q10 零压降、U31 精确）。
- 欠压/过压：UV<15V 关断、OV>33V 关断，保护后级隔离模块与 MCU。

**验证点**：HW-PWR-002（可复位过载）、HW-PWR-003（短路保护）、HW-PWR-005（PGOOD/FAULT）。

## 第 6 级 U30 — 隔离 DC-DC（电气隔离边界）

| 项 | 值 |
|----|-----|
| 器件 | U30 = 金升阳 URB2424P-30WR3（24V→24V，30W，隔离 ≥1500Vdc，效率 ~85%） |
| 输入 | C30 = 22µF 陶瓷（初级侧，接 NET_24V_IN / NET_GND_IN） |
| 输出 | C32 = 22µF 陶瓷 + 100µF 电解（次级侧，NET_24V_BOARD / NET_GND） |
| 隔离 | 初次级间 ≥4mm creepage；次级地 NET_GND 与初级 NET_GND_IN 完全浮空 |

**机理**：这是**系统级隔离边界**——将 24V 输入域（可能带动力地噪声/地电位差）与控制器域彻底隔开，
既消除地环路，又使后续 MCU/总线对输入侧共模瞬变免疫。额定 30W ≫ 板载峰值 ~8W，裕量充足。

**验证点**：HW-PWR-004（隔离耐压 ≥1.5kV）。

## 第 7 级 次级供电 — Buck + LDO

| 项 | 值 |
|----|-----|
| U33 | TPS54331（Buck，24V→5V，3A，SOP8）；L33=10µH，FB 分压 R33=30.1k/R34=10k → 5.0V |
| U34 | AMS1117-3.3（LDO，5V→3.3V，0.8A，SOT223）；C36/C37 钽/陶瓷 |
| 直接 24V 负载 | 传感器（部分 24V）、LED、蜂鸣器、安全继电器线圈由 NET_24V_BOARD 经本地限流供给 |

**机理**：隔离后的 24V 既直供现场传感器/执行指示，又经 Buck→LDO 得到干净的 5V/3.3V 逻辑电源；
3.3V 为 MCU/收发器逻辑侧主电源，5V 供 PHY/RJ45 磁芯与隔离收发器总线侧（经 B0505S）。

## 逐级验证结论（设计态）

| 测试 | 级别 | 机理 | 设计通过判据 |
|------|------|------|--------------|
| HW-PWR-001 防反 | Q10 | P-MOS 体二极管反偏阻断 | 反接时 NET_24V_BOARD=0V、无发热；正接恢复后正常 |
| HW-PWR-002 过载 | U31 | 主动限流+自动重试 | 加载 2.2A，U31 限流至≈2A 且不锁死，FLTb 拉低 |
| HW-PWR-003 短路 | U31 | 限流+热关断重试 | 输出短路，电流被钳、无烧毁，移除后自恢复 |
| HW-PWR-004 隔离 | U30 | ≥1500Vdc 隔离 | 初次级间 1.5kVdc/1min 无击穿（出厂/抽检） |
| HW-PWR-005 PGOOD/FAULT | U31 | PGOODb/FLTb 状态 | 正常 PGOODb=高；过流 FLTb=低且 MCU 可读 |

> 设计态结论：链序与器件参数在原理上满足 5 项；**实测需在样机阶段执行**（见 `../docs/06_testpoints.md`）。
