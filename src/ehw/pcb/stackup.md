# 层叠与布局约束（2 层板 100×80mm）

> 约束权威源。Layout 须满足本文件；禁用 4 层，无内层电源/地平面。

## 1. 层叠结构（2 层）

| 层 | 名称 | 用途 |
|----|------|------|
| Top | L1 | 信号 + 局部电源（24V/5V/3.3V 走线）+ 地铜皮（非完整平面） |
| Bottom | L2 | 信号 + 地铜皮（主要回流路径）+ 少量电源走线 |

- 板厚 1.6mm ±0.1；材质 FR-4（TG≥135）；表面 ENIG（无铅，适合细间距 LQFP176 与模块焊盘）。
- 走线宽/间距：信号 0.2mm/0.2mm；电源 24V ≥0.6mm（2A 载流）；5V ≥0.4mm；3.3V ≥0.3mm。
- 过孔：0.3mm 孔 / 0.6mm 盘（标准）；大电流过孔阵列（24V/5V 通流处 ≥3 个并联）。
- 铜厚 1oz（35µm）；高电流路径（24V 输入→U30）可局部加厚或露铜加锡。

## 2. 地策略（2 层无完整平面）

- **初级地 NET_GND_IN** 与 **次级地 NET_GND** 在板面物理分离，分割槽 ≥4mm（creepage）；二者仅经 U30 隔离边界连接（无铜连接）。
- 次级地采用「Top/Bottom 双铜皮 + 密集过孔阵列」构成低阻抗回流；MCU 下方地铜皮单点经 0Ω/磁珠连接模拟地。
- 所有高速信号（RMII 50MHz、CAN、SDIO）下方保留连续地铜皮，禁止跨分割。

## 3. 关键间距/安规

| 项 | 要求 |
|----|------|
| 初级↔次级爬电 | ≥4mm（U30 两侧铺铜不跨区） |
| 24V 线间距 | ≥0.5mm（功能绝缘） |
| 高压/敏感模拟 | 远离数字/时钟 ≥3mm |
| 隔离收发器总线侧 | 其隔离地 NET_GND_ISO 与逻辑地 NET_GND 分割 ≥2mm |

## 4. 拼板/工艺

- 样机：单板 100×80；可 2-up 拼板（间距 5mm，V-cut）。
- 丝印：白色，见 `docs/04_silkscreen.md`；不满足处用阻焊丝印（白油）补。
- DFM：所有封装符合 IPC-7351；LQFP176 推荐钢网开窗 1:1，QFN24 露窗防桥连。
