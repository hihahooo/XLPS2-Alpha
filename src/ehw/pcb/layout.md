# PCB 布局与布线规则

## 1. 分区布局（Top 视图，100×80mm）

```
 +--------------------------------------------------+
 | [J1 24V] [L30][Q10][F1][U31]==[U30隔离]          |  ← 初级/隔离区(上)
 | ===== 4mm 分割槽(初/次极) ======================= |
 | [U33 Buck] [U34 LDO] [U4+J14 ETH]               |
 | [U2A/U2B ISO1042][U40-43 B0505S]  [J2-J5]       |  ← 接口区(右)
 | [U1 STM32H743]  [U6/U7 I2C]  [U5 WS63]          |  ← MCU/无线区(中)
 | [U9/Q11/K1 安全] [LED/BUZ] [J6 SWD][J13]        |  ← 安全/调试区(下)
 +--------------------------------------------------+
```

## 2. 布局要点

1. **电源链顺序不可绕**：J1→L30→Q10→F1→U31→U30 沿板边一字排开，走线短直；U30 后跨分割进入次级区。
2. **隔离边界**：U30 下方留 ≥4mm 净空，初次级铜皮不相交；B0505S 紧邻对应隔离收发器。
3. **MCU 居中**：STM32H743 居中，去耦 100nF 围绕四周 <5mm；VCAP 紧靠。
4. **时钟隔离**：Y1(25MHz) 紧靠 PH0/PH1，下方禁止走线，包地；远离模拟/电源。
5. **ETH RMII**：U4 紧靠 J14 与 U1 的 RMII 引脚；RMII 差分/50MHz 等长 ≤2mm 偏差，包地。
6. **无线 WS63**：U5 板边布置，天线区域禁铺铜/走线（净空 ≥5mm）；SDIO 走线等长。
7. **安全回路**：U9/Q11/K1 靠近板边连接器，路径最短；K1 触点走大电流不穿 MCU 区。

## 3. 布线规则

- 电源：24V→U30 用铺铜（≥2mm 宽或露铜）；5V/3.3V 星型走线，过流处多过孔。
- 高速：RMII 50MHz、SDIO(≤50MHz) 做阻抗≈50Ω 单端，包地，少换层。
- CAN/RS485：差分对（CANH/L、A/B）紧耦合，终端电阻靠近连接器；TVS 最靠近端口。
- 模拟：VDDA 磁珠隔离，VDD/VSS 退耦贴近引脚；ADC 输入走线短、远噪声。
- 所有 GND 经 Bottom 铜皮 + 过孔阵回流；禁止信号线跨地分割（除隔离边界外）。

## 4. 测试点与丝印锚点

- 每级电源设测试点（TP1~TPn，见 `docs/06_testpoints.md`）：NET_24V_IN/PROT/EF/ISO/BOARD/5V/3V3/GND_IN/GND。
- 关键信号设探针点：FDCAN1/2、USART3/6、RMII_REF_CLK、SAFE_HEARTBEAT。
