# EHW 嵌入式硬件架构与开发说明（真实设计文档）

> 全新开发基线。**不复用** NP@XLPS2 历史设计文件，仅沿用其沉淀的可落地思路。
> 设计产物见 `src/ehw/`（原理图源 / 网表 / 八文档 / BOM / 引脚表 / 封装库）。本文为模块总说明。

## 1. 职责

控制器硬件平台：24V 隔离电源链、STM32H743 最小系统、通信接口（隔离 CAN-FD/RS485/RJ45/WS63）、
外部存储、安全回路物理基础。为 CFW 可升级（A/B）与现场抗扰预留冗余。

## 2. 关键规格

| 项 | 规格 |
|----|------|
| MCU | STM32H743IIT6（LQFP176，Cortex-M7 400MHz，2MB Flash，1MB SRAM，内置 ETH MAC） |
| 板型 | **2 层 PCB 100×80mm**，1.6mm ENIG；禁用 4 层，无内层平面 |
| 输入 | DC 24V（动力 48V 由机柜另供，不经本板） |
| 隔离 | 系统级 U30(≥1.5kV) + 总线级 ISO1042/ADM2483 + B0505S；RJ45 磁芯隔离 |
| 存储 | 内部 Flash 2MB（A/B+FLAG，ADR-005）；外挂 FRAM FM24CL64（干扰点库，I2C）+ EEPROM 24C64（标定） |
| 无线 | WS63（WiFi/星闪/BLE，SDIO+UART4） |
| 温度 | 系统 −30~+60℃；器件级更宽 |

## 3. 24V 输入保护链（权威顺序，全文档一致）

```
J1(DC24V_IN)
 → L30(共模/π滤波 10µH/3A)
 → Q10(防反 P-MOS AO3401A + D31 BZT52C12 钳 Vgs)
 → F1(PPTC 2A 自恢复)
 → U31(eFuse TPS26600 可复位 OCP/SCP，R_ILIM=6.04kΩ(1%) → ≈1.95A)
 → U30(隔离 24V→24V DC-DC 金升阳 URB2424P-30WR3 ≥1.5kV)
 → NET_24V_BOARD → Buck(U33 24→5V)→LDO(U34 5→3.3V) 及 传感器/LED/蜂鸣器
```

- **地分割**：NET_GND_IN（U30 前）vs NET_GND（U30 后）≥4mm creepage，仅经 U30 隔离连接。
- 链序**绝对不可颠倒**（防反→保险→eFuse→隔离）。逐级机理与 HW-PWR-001~005 见 `src/ehw/sch/01_power_chain.md`、`src/ehw/docs/06_testpoints.md`。

> ✅ **已裁决确认（主理人 #2）**：原文档「R_ILIM=24k ~2A」经 TI TPS26600 数据手册公式
> **I_LIM(A)=11.8/R_ILIM(kΩ)** 核算，24kΩ 仅 ≈0.49A，与「2A」矛盾。主理人裁定目标限流 ≈2A，
> 采纳 **R_ILIM=6.04kΩ(1%)**（I_LIM≈11.8/6.04≈1.95A）。本设计维持 6.04kΩ，无需改动。

## 5. 连接器与接口

- `J1`=24V 电源输入；`J14`=以太网 RJ45 **HR911105A**（带磁芯，禁 HR601680）；`J13`=扩展排针 2x10。
- 隔离 CAN-FD ×2（J2/J3，伺服 D_00/D_01）；隔离 RS485 ×2（J4/J5，激光/网关 Modbus RTU）。
- 调试 SWD（J6 1.27 5P）。接口引脚定义见 `src/ehw/docs/07_interfaces.md`。

## 6. 安全回路

急停/安全触边 **硬件直切动力**（ESTOP/EDGE NC 串联 → 光耦 → 使能 FET → 安全继电器 K1 → 伺服使能链），
**独立于 MCU 软件**；MCU 安全心跳（IWDG 兜底）驱动使能，固件挂死即自动切除。
对应 HSM「区域3 安全监控区（region=2，最高优先级）」，符合 ISO 3691-4。详见 `src/ehw/sch/04_safety.md`。
